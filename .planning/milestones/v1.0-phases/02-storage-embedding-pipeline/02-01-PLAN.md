    pub struct BreakPoint {
        pub pos: usize,    // byte position
        pub score: f64,    // higher = better cut (h1=100, h2=90, h3/codeblock=80, h4=70, h5/hr=60, h6=50, blank=20, list=5, newline=1)

    **`pub fn scan_break_points(text: &str) -> Vec<BreakPoint>`**

    Walk the bytes looking for `\n` and classify what follows. At each position keep only the highest score if multiple patterns match. Return sorted by position. Score table (match against bytes after the `\n`):
    - `#` not followed by `#` → h1, 100
    - `##` not followed by `#` → h2, 90
    - `###` not followed by `#` → h3, 80
    - `\`\`\`` (three backticks) → codeblock, 80
    - `####` not followed by `#` → h4, 70
    - `#####` not followed by `#` → h5, 60
    - horizontal rule (`---\n`, `***\n`, `___\n` at the newline) → hr, 60
    - `######` → h6, 50
    - another `\n` (blank line) → blank, 20
    - `- ` or `* ` (list item) → list, 5
    - digit + `.` + space (ordered list) → numlist, 5
    - any other `\n` → newline, 1

    Heading patterns: count consecutive `#` chars; h-level = count; score = 110 - count*10 (capped at h6=50). Skip if `#` count > 6. A `#`-sequence is a heading only if immediately followed by a space or end of text.

    **`pub fn find_code_fences(text: &str) -> Vec<CodeFenceRegion>`**

    Scan for `\n\`\`\`` (newline + three backticks). Toggle in/out of fence on each match. If unclosed at end, extend region to `text.len()`.

    **`pub fn is_inside_code_fence(pos: usize, fences: &[CodeFenceRegion]) -> bool`**

    `fences.iter().any(|f| pos > f.start && pos < f.end)`

    **`pub fn find_best_cutoff(breakpoints: &[BreakPoint], target: usize, window: usize, decay: f64, fences: &[CodeFenceRegion]) -> usize`**

    From qmd: iterate break points in range `[target - window, target]`. For each, skip if inside a code fence. Compute adjusted score using squared distance decay:
    ```
    normalized_dist = (target - bp.pos) as f64 / window as f64
    multiplier = 1.0 - normalized_dist * normalized_dist * decay
    final_score = bp.score * multiplier
    ```
    Return position of highest `final_score`. If none found, return `target`.

    **`fn chunk_by_size(content: &str) -> Vec<(usize, usize)>`** — Returns `(start, end)` byte positions.

    ```
    Pre-compute: breakpoints = scan_break_points(content), fences = find_code_fences(content)
    If content.len() <= CHUNK_SIZE_CHARS: return vec![(0, content.len())]
    Loop:
      char_pos = 0
      while char_pos < content.len():
        target_end = min(char_pos + CHUNK_SIZE_CHARS, content.len())
        if target_end < content.len():
          best = find_best_cutoff(breakpoints, target_end, CHUNK_WINDOW_CHARS, DECAY_FACTOR, fences)
          if best > char_pos && best <= target_end: end_pos = best else: end_pos = target_end
        else:
          end_pos = content.len()
        push (char_pos, end_pos)
        if end_pos >= content.len(): break
        next_start = end_pos.saturating_sub(CHUNK_OVERLAP_CHARS)
        // Ensure forward progress
        if next_start <= char_pos: char_pos = end_pos else: char_pos = next_start
    ```

    The heading_byte_offset should be the byte offset of the `\n` before the `#` (i.e., the start of the heading line), so that a chunk starting exactly at a heading catches it.

    **Pass 2 — Smart chunk the content-after-frontmatter:**
    ```rust
    let stripped = &content[frontmatter_end_byte..];
    let positions = chunk_by_size(stripped);  // (start, end) relative to stripped
    ```

    - `line_end = byte_offset_to_line(content, abs_end.saturating_sub(1))`
    - Create `Chunk { file_path, heading_breadcrumb, heading_level, body, line_start, line_end, frontmatter: frontmatter.clone() }`
