### IN-01: Force-exit uses code 1 instead of conventional 130

**Files modified:** `src/daemon/shutdown.rs`
**Commit:** 947b29f
**Applied fix:** Changed `std::process::exit(1)` to `std::process::exit(130)` on the force-exit path. Exit code 130 (128 + SIGINT signal number 2) is the POSIX convention for a process terminated by SIGINT, making forced exits distinguishable from application errors in shell scripts and automation.
