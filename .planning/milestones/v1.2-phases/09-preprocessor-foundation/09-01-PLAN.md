Define:

```rust
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum PdfClassification { TextFirst, NeedsVision }

pub fn classify_pdf(bytes: &[u8], max_bytes: usize) -> Result<PdfClassification, LocalIndexError>;
pub fn extract_text_pdf_as_markdown(bytes: &[u8], max_bytes: usize) -> Result<String, LocalIndexError>;
```

**Rules (concrete):**
- If `bytes.len() > max_bytes`, return error `LocalIndexError::...` with message containing substring `asset too large`.
- Use `lopdf` to open in-memory document; iterate pages; extract text streams where possible; concatenate with `\n\n` between pages.
- **Classification:** `TextFirst` if total extracted printable characters ≥ `12 * page_count` (or page_count inferred from page tree; minimum 1 page); else `NeedsVision`. Document magic numbers in doc comments.

Unit test: construct minimal valid PDF bytes **inline** or from a tiny `tests/fixtures/minimal_text.pdf` (add file &lt; 8KiB) — assert `classify_pdf` returns `TextFirst` and markdown contains a known substring.

If generating minimal PDF in-test is too brittle, commit `tests/fixtures/phase09/hello.pdf` with text "PHASE09_FIXTURE" and assert substring match.
