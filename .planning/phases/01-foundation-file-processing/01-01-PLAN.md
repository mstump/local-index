    **src/cli.rs** -- Create clap derive structs per CLI-06 and CLI-07:

    `Cli` struct with `#[derive(Parser, Debug)]` and `#[command(name = "local-index", about = "Semantic search over local markdown vaults", version)]`:
    - `vault_path: PathBuf` -- `#[arg(long, env = "LOCAL_INDEX_VAULT_PATH", default_value = ".")]`
    - `db_path: PathBuf` -- `#[arg(long, env = "LOCAL_INDEX_DB_PATH", default_value = ".local-index")]`
    - `log_level: String` -- `#[arg(long, env = "LOCAL_INDEX_LOG_LEVEL", default_value = "info")]`
    - `command: Command` -- `#[command(subcommand)]`

    `Command` enum with `#[derive(Subcommand, Debug)]`:
    - `Index(IndexArgs)` -- doc comment: `/// Index all markdown files in the vault (one-shot)`
    - `Daemon(DaemonArgs)` -- doc comment: `/// Start daemon: watch files + serve HTTP`
    - `Search(SearchArgs)` -- doc comment: `/// Search the index`
    - `Status` -- doc comment: `/// Show index status`
    - `Serve(ServeArgs)` -- doc comment: `/// Start HTTP server only (no file watcher)`

    `IndexArgs` struct:
    - `force: bool` -- `#[arg(long, help = "Force re-index all chunks, ignoring content hashes")]`

    `DaemonArgs` struct:
    - `port: u16` -- `#[arg(long, env = "LOCAL_INDEX_PORT", default_value_t = 3000)]`
    - `bind: String` -- `#[arg(long, env = "LOCAL_INDEX_BIND", default_value = "127.0.0.1")]`

    `SearchArgs` struct:
    - `query: String` -- positional arg, doc: `/// Search query text`
    - `limit: usize` -- `#[arg(short = 'n', long, default_value_t = 10)]`

    `ServeArgs` struct:
    - `port: u16` -- `#[arg(long, env = "LOCAL_INDEX_PORT", default_value_t = 3000)]`
    - `bind: String` -- `#[arg(long, env = "LOCAL_INDEX_BIND", default_value = "127.0.0.1")]`

    Add a unit test module `#[cfg(test)] mod tests` with:
    - `test_cli_parse_index`: verify `Cli::try_parse_from(["local-index", "index"])` succeeds
    - `test_cli_parse_search`: verify `Cli::try_parse_from(["local-index", "search", "hello"])` succeeds and query == "hello"
    - `test_cli_default_values`: verify default vault_path is ".", db_path is ".local-index", log_level is "info"

    **src/logging.rs** -- Create tracing initialization per CLI-08:

    `pub fn init(log_level: &str) -> anyhow::Result<()>` that:
    1. Creates `EnvFilter` from `RUST_LOG` env var (via `EnvFilter::try_from_default_env()`), falling back to the `log_level` parameter
    2. Builds `tracing_subscriber::registry()` with the filter and a `fmt::layer().with_target(true).with_thread_ids(false)`
    3. Calls `.init()`
    4. Returns `Ok(())`

    Use these imports: `tracing_subscriber::{fmt, EnvFilter, layer::SubscriberExt, util::SubscriberInitExt}`

    **src/types.rs** -- Create data structures for chunks and frontmatter:

    `Frontmatter` struct with `#[derive(Debug, Clone, Default, Serialize, Deserialize)]`:
    - `tags: Vec<String>` with `#[serde(default)]`
    - `aliases: Vec<String>` with `#[serde(default)]`
    - `title: Option<String>` with `#[serde(default)]`
    - `date: Option<String>` with `#[serde(default)]`
    - `extra: HashMap<String, serde_yml::Value>` with `#[serde(flatten)]`

    `Chunk` struct with `#[derive(Debug, Clone, Serialize)]`:
    - `file_path: PathBuf`
    - `heading_breadcrumb: String`
    - `heading_level: u8`
    - `body: String`
    - `line_start: usize`
    - `line_end: usize`
    - `frontmatter: Frontmatter`

    `ChunkedFile` struct with `#[derive(Debug)]`:
    - `file_path: PathBuf`
    - `frontmatter: Frontmatter`
    - `chunks: Vec<Chunk>`

    **src/error.rs** -- Create error types using thiserror:

    `LocalIndexError` enum with `#[derive(Debug, thiserror::Error)]`:
    - `#[error("Chunk error: {0}")]` `Chunk(String)`
    - `#[error("Walk error: {0}")]` `Walk(#[from] walkdir::Error)`
    - `#[error("IO error: {0}")]` `Io(#[from] std::io::Error)`
    - `#[error("YAML parse error: {0}")]` `YamlParse(String)`
    - `#[error("Config error: {0}")]` `Config(String)`

    After creating all four files, run `cargo test --lib` to verify compilation and unit tests pass.
