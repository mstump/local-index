  <action>
    Update `src/main.rs` to implement the `Command::Index` match arm:

    1. Resolve vault path: `let vault_path = cli.vault_path.canonicalize().map_err(|e| anyhow::anyhow!("Invalid vault path '{}': {}", cli.vault_path.display(), e))?;`
    2. Log start: `tracing::info!(path = %vault_path.display(), "starting index of vault")`
    3. Call `local_index::pipeline::walker::discover_markdown_files(&vault_path)` to get file list
    4. If file list is empty: `tracing::warn!("no markdown files found in vault")` and return Ok(())
    5. For each file path:
       a. Read file content with `std::fs::read_to_string(&file_path)?`
       b. Compute vault-relative path: `file_path.strip_prefix(&vault_path).unwrap_or(&file_path)`
       c. Call `local_index::pipeline::chunker::chunk_markdown(&content, &relative_path)?`
       d. Log per-file: `tracing::info!(file = %relative_path.display(), chunks = chunked_file.chunks.len(), "chunked file")`
       e. Accumulate total chunks count
    6. Print summary to stdout: `println!("Indexed {} files, {} chunks", file_count, total_chunks)`
    7. For each chunk across all files, print a JSON line to stdout using `serde_json::to_string(&chunk)?` (one JSON object per line, JSONL format). This enables piping output to other tools.
    8. Log completion: `tracing::info!(files = file_count, chunks = total_chunks, "indexing complete")`

    Keep the `todo!()` stubs for Daemon, Search, Status, Serve.

    Handle errors per file gracefully: if a single file fails to read or chunk, log `tracing::warn!(file = %path.display(), error = %e, "failed to process file")` and continue to next file (do not abort the entire walk).
